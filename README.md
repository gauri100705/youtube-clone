📺 YouTube Homepage Clone

A pixel-inspired YouTube homepage clone built using pure HTML5 & CSS3.
This project focuses on mastering layout structuring, positioning, hover interactions, and UI replication.

🌐 Project Overview

This is a front-end clone of the YouTube desktop homepage.
The goal was to recreate the layout structure, navigation system, and visual styling as closely as possible using only HTML and CSS — without JavaScript.


It demonstrates strong understanding of:

Layout alignment

Fixed positioning

Flexbox & grid structuring

Hover effects & tooltips

Sidebar navigation design

Clean UI structuring


✨ Features

✔ Fixed header with logo, search bar & action icons
✔ Functional sidebar layout
✔ Sidebar footer section (YouTube-style integrated links)
✔ Video grid layout
✔ Hover tooltips
✔ Notification badge styling
✔ Responsive structure foundation

HTML5

CSS3
